# 📁 Sample Images Directory

## Bienvenue ! 👋

C'est ici que tu mets tes images de test pour LunaLens !

## Comment utiliser

1. **Ajoute tes images ici** — JPG, PNG, GIF, BMP, WebP sont les bienvenus
2. **Lance LunaLens** — Exemple:
   ```bash
   python -m src.cli analyze data/sample_images/ton_image.jpg
   python -m src.cli ascii data/sample_images/ton_image.jpg
   python -m src.cli insights data/sample_images/ton_image.jpg
   ```

## Idées d'images à tester

- 🌅 **Paysages** — couchers de soleil, montagnes, plages
- 🐱 **Animaux** — mes amis à fourrure préférés
- 🎨 **Art** — peintures, sculptures, créations
- 🏙️ **Urbain** —'architecture, rues, lumières
- 🌸 **Nature** — fleurs, feuilles, textures
- 📸 **Portraits** — visages et expressions

## Tips

- Les images carrées ou paisibles fonctionnent mieux pour l'ASCII art
- Les images avec bon contraste donnent de meilleurs résultats
- Plus l'image est grande, plus l'ASCII art sera détaillé !
